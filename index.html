Welcome to My Helm Chart Repository

This repository contains Helm charts for various Kubernetes applications. Below is a quick guide on how to install the charts from this repository.

Installation Guide

  1. Ensure you have Helm installed on your machine.
  2. Add the Helm chart repository to your Helm client:
    3. helm repo add my-repo https://kanechoo.github.io/chart-repo/charts
  3. Update your Helm repository list:
    4. helm repo update
  4. Search for available charts in the repository:
    5. helm search repo my-repo
  5. Install a specific chart from the repository. Replace `` with the name of the chart you want to install:
    6. helm install my-release my-repo/
